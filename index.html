四、Basic SOP

（1）导流违规

  1. 商家自行申诉
  2. 自行申诉失败，找百科全薯代客诉

⚠️ 没有其他解罚手段，导流违规是平台红线，不可触碰！

（2）直播限流、店铺违规

  1. 让商家发账号ID和店铺ID
    • 提取账号UID：通过【APP - 我 - 右上角"..."- 复制链接】粘贴出来即可
    • 提取店铺方法：千帆后台【我的】找
  2. 找hi上的【社区审核百科全薯】右边组件，查账号违规情况
  3. 找【直播审核值班服务号】，问是否有违规导致直播间限流

Tips：所有有关商家咨询流量不好、违规等行为都可以通过该方式进行查询，如果没有解决或者想查询详细信息可以看【五-(1)店铺信息查询】。

（3）获取账号与店铺ID

账号ID获取方式：点击主页右上角 ➡️ 选择复制链接 ➡️ 转发到聊天框 ➡️ 点击链接 ➡️ 复制链接 ➡️ 再转发到聊天框 ➡️ 截取ID这一部分（位于 profile/ 和 ? 之间的这串字符，就是所需ID）

店铺ID获取方式：点击店铺，选择右上角...同上

ps. 建议直接让商家发链接，由小助手截取ID

（4）商家反馈流量不好

  1. 提高笔记发布频次，绑定直播预告 — 提升账号整体曝光
  2. 直播间调整封面图片以及标题，更吸睛更突出利益点 — 提升直播间点击进房
  3. 直播间设置福利款漏价款、定点放漏 — 提升直播间留人率
  4. 每个整点自行补单，次日再退 — 优化直播间数据，提升算法推流

Tips：想做店铺诊断可以看【五-(2)】店铺数据分析，这个属于进阶技能

（5）店铺主体变更、更换营业执照

  1. 需要收集商家的信息（发这段话术，一直到注意结束）：

    【主体变更入口开白】提前给我以下信息，我这边走特批申请，中台固定每周二操作打开主体变更的入口；

    • 账号ID（userid）
    • 店铺ID
    • 店铺名称
    • 变更理由

    注意：

    1. 变更当月不能操作提现，如果已经提过现则下个月再做申请
    2. 开白入口后按照链接里的流程步骤操作就行（参考小红书电商学习中心）
  2. 申请商家登记链接：二奢主体变更商家登记
  3. OA申请链接：Redflow

    备注：

    • 检查账号ID和店铺ID等信息后填OA
    • 特批类型：入驻后：店铺主体变更
    • 店铺主题变更开白场景选择：跳过店铺类型校验
  4. 一周如果没有反馈，可以填催审登记表
  5. 审核通过后发送：【审核通过了，可以按照链接继续操作变更了】

ps. 更换营业执照就要走这个流程

（6）怎么联系商服

APP端找客服路径：小红书APP - 右下角"我" - 左上角三栏 - 专业号中心 - 右上角"耳机"图标 - 发送"人工" - 点击"需要客服帮助" - 点击"联系人工" - 联系人工后选择相对应的问题即可跳转到对应客服

PC端找商服路径：登陆商家管理后台（千帆） - 右下角"帮助"图标 - 即可联系商服进行沟通

（8）类目邀约申请（定邀）

老店入驻

是否符合定邀资质需要check两个点：

  1. 是否满足个体店或普企店（个人店不允准入，尽快去做主体升级）
  2. 在BI查他的二奢类目近30天dgmv是否≥5w

新店或原其他类目

想做二奢就必须满足要求的资质（店铺类型和二奢类目交易门槛），同时商品挂到二奢类目底下走QIC鉴定。看看商家是否接受，若接受走下面的具体流程

新店具体流程

  1. 先确认一下店铺、专业号是否完成操作
  2. 先看文档，里面的步骤很清楚，具体的准入要求都在这里了，先申请，然后把补充信息和营业执照发给我，我这边帮您通过
  3. 登陆鹰眼检索商家名称完成定邀申请
  4. 提示商家手动添加新类目（商家很容易忘！！！），然后进新的群聊，告诉他们留意群消息，重要通知都在群里
  5. 将信息登记在：登记表单1、登记表单2（现在使用）
  6. 3天登陆奥汀OA，看一下是否有被驳回的，如果有，按照要求提示商家进行修改

⚡ 先给一点提示：

  • 流程：先看店铺信息、主营类目是否为二奢 ➡️ 输入信息、填写登记表单 ➡️ 询问其他经营平台情况 ➡️ 提交OA ➡️ OA查看未通过名单
  • 要严格进行商家背调，包括但不限于收集商家仓库照片、其他平台链接、其他平台GMV截图、是否有违规记录等
  • 在填写表单的时候注意用词，不要出现让人一看看不懂的词语，直白语言表述
  • 表单中填写的申请ID是上一个界面生成的5位短ID

📋 表单填写的时候注意：

  • 无其他经营平台，放【商家货架、仓库】必须有2张实体照片+证明
  • 有其他经营平台且是同一主体，GMV放到【近6个月其他平台】，不需要写备注
  • 有其他经营平台且是关联主体，GMV+关联证明放到【近6个月其他平台】这个位置，然后写：【备注：...】
  • 有其他经营平台且是无关主体，GMV放到【近6个月其他平台】，然后写：【备注：...】
  • 最下面要填写两个ID，然后到苍穹截取店铺图片补充到下面，其他特殊图片也放到这里

（9）QIC加白与取消

QIC开白每两周进行一次，具体需求、话术如下：

为了提高二奢重点商家经营效率、发货&履约时效、优化商家及用户体验，我们特向平台申请了二奢QIC白名单，进入白名单的商家无需再强制过平台QIC单单质检。前提是商家自身有自主送检能力，有长期稳定合作的质检机构，且同时满足门槛条件。

白名单门槛及指标如下，我们会双周维度滚动跑数据，如果后续指标不达标，则会被自动剔除白名单，恢复单单质检：

  1. 主营类目为「二手奢侈品」
  2. 15天订单量≥10
  3. 14天正品质疑（就是假货售后）=0
  4. 近30天GMV≥20万
  5. 店铺分≥4.6
  6. 具备自主送检能力

咱家目前是满足门槛条件的，但也充分考虑到商家的自主意向，可以选择申请白名单，也可以继续走平台QIC流程，加白后就全店免检了，没办法按订单维度选择是否送检。要送检的话只能你们自己线下送，线上就不会同步鉴定流程和结果了。

【请回复：是否申请加白免检】

注意事项

  • 目前为止（2026.1.30）开白店铺没有标识（后续更新）
  • 生效时间为当日，但一般反馈为下周一
  • 商品列表不需要更改
  • 取消开白的名单统计到表格中，同时上报给奥汀，奥汀会反馈给【治理】

ps：名单两周一次，目前在讨论是否会取消QIC加白，等待ing（时间戳-2026年2月10日）

（10）投广&投流

  1. 流量卡可以自己在千帆后台参与活动任务，完成任务会发放
  2. 商业化投广需要找商业化直客销售提供资质材料，走特批开白账户后才可以有投广资质。需要是企业主体、注册时间不少于三年，以及其他条件。都满足的话可以找商业化老师开白认证回收/售卖资质
  3. 我们（运营侧）扶持10w+场次，可以盘一下是否能冲刺单场10w+，如果可以的话就统计上报大场

ps：当然一般可以冲击10w场次的也不用统计上报，周会自然会筛选出来

（11）10w+大场投流（旧）

  1. 统计时间：每个月的大场在上个月月末统计
  2. 统计人员：全体二奢商家（重点B4、B5、B6）
  3. 具体流程：
    1. 登陆BI导出最新商家层级表
    2. 用腾讯文档制作大场统计表
    3. 转发到商家群，提醒商家填写
    4. 先粘贴到大场导入表（需要复制下来存到电脑中），然后导入到系统（苍穹）
    5. 如果有导入失败按照要求修改

提示：需要用到账号ID和主播ID，商家可能不会查询

（12）负反馈处置

  1. 核实负反馈情况是否属实，是店铺导致的还是商家导致的
  2. 处置判定与文档：品质负反馈处置咨询必读
  3. 判定属于【恶意差评】影响负反馈数据的，找 @植木(成芊林) 申请解除

（14）大场投流

大场投流包括两部分，一个是【直播卡】投流，一个是【笔记】投流，每天都需要做以下几个动作

⏰ 操作时间：每天17:00操作，周五上周六、周日、周一的数据

步骤一：【苍穹 - 直播运营 - 直播计划盘点】

Tips：这部分会关联到【苍穹 - 直播运营 - 直播日历】，可以查看商家的达成情况

  1. 奥汀账号打开操作平台 👉 苍穹
  2. 筛选第二天的直播信息，选择【待审批】、【待开播】、【奥汀】三个字段，金额根据最近活动要求，时间选择今天到后天
  3. 看商家信息，预判是否可以达成目标，如果可以就通过，如果觉得无法实现，就私信商家核实情况

步骤二：【曙光 - 直播计划】（笔记投流）

  1. 操作位置 👉 曙光
  2. 如图所示选择字段，【自定义时间】为今天到后天，主播ID填写重点主播ID名单（用英文逗号间隔），注意设置GMV范围，其他字段根据实际情况选择
  3. 只有创建直播计划才可以进行预告投放
  4. 日期从当前开始，时间越长投放效果越好，一般不超过3天；投放量为上限的三分之一；笔记数填最大值
  5. 填写完成后导出第二天的直播计划，待会儿第三步要用到

步骤三：【曙光 - 流量扶持 - 投流计划】（直播卡投流）

  1. 打开链接 👉 曙光
  2. 点击批量创建
  3. 填写前四列字段：
    • 主播ID、开始时间从上一步下载的数据中复制粘贴
    • 结束时间为当天23:59:59
    • 预算小组为：扶持：电商行业/二奢
  4. 建议此处先导入商家名单，操作见【f】步骤
  5. 下载结果查看未成功原因
  6. 若显示【主播不存在】，在【曙光 - 流量扶持 - 投流名单】中添加名单，名单就是上面第二步导出的商家ID，用空格分开，直接导入

Tips：不要超过200条；每天上传第二天的

五、Advanced SOP

（1）店铺信息查询

当商家询问是不是有限流、违规的时候，可参考以下步骤进行店铺诊断：

  1. 询问账号ID
  2. 用【审核百科全薯】的右侧工具栏查询
    • 若商家非导流问题可以进行申诉，等待处理结果
    • 若结果不可查看步骤3和4
  3. 上【鹰眼】做诊断（店铺诊断）
    • 重点看店铺违规分析和直播分析
    • 黑盒处理不可透传，部分可以整改，有选择性的反馈给商家
    • 结果无法查询，带账号信息戳【审核百科全薯】
  4. 上【曙光】查询直播间（直播违规）
    • 在【审核处置记录】中看是否有违规信息
    • 曙光查询大部分是违规导流，导流处置跳转【四-(1)初次导流违规】
    • 结果无法查询，带账号信息戳【审核百科全薯】，可能需要跳转【大商业-交易审核值班号】

（2）店铺数据分析

BI的数据助手可以对商家的店铺、账号、直播间从各维度进行分析，按需取用。但是他最后输出的分析不一定都对商家有价值，你要拣重点 & 转化成人话告诉商家该从哪些方面、如何优化。

六、Technical Terms

专业术语速查表

BT

Brandtag，常记录为"BT纬度"。是指商业或营销分析中，用于衡量和定位品牌、产品的一个"维度"。可以被理解为一种专为品牌营销和用户管理而设计的标签体系。

闭环电商

是当前内容平台商业化的核心战略方向，特指在同一个平台或生态体系内，完成从"内容种草"到"交易转化"的全过程，构建一个完整的用户消费路径与数据追踪闭环。

case

需要分析处理的"案例"或"问题样本"。

CPM

Cost Per Mille 的缩写，意为"千次展示成本"。它衡量的是广告每获得一千次曝光（展示）所花费的费用。

CTR

Click-Through Rate 的缩写，即"点击率"。它衡量的是看到广告的人中，有多大比例被吸引并点击了广告。

DGMV

日交易总额。

店播

Store Broadcast / Brand Live，指品牌或商家自己开设直播间，由品牌员工或专属主播进行直播。这是品牌的"自营渠道"。

达播

KOL Broadcast，指与达人/博主（KOL/KOC）合作，也叫K播。在其个人或机构的直播间进行产品推广和销售。

动销

全称是"商品动销率"，是衡量零售业务健康度的一个核心指标。指在一定时间段内，有实际销售出库记录的商品品种数，占同期总库存商品品种数的比例。

灰度

分阶段、小范围、可控的产品发布或测试策略。这是一种风险控制机制。

involve

强调"必然关联"或"必需条件"，involve某人表示的就是要和某人一起达成某种目的，需要和某人建联。

KFS

K（Key Opinion Leader）+ F（Feed）+ S（Search）三种核心流量获取与转化方式的组合策略。将内容种草、付费放大和主动搜索承接结合起来，形成一个高效的营销闭环。

KSO

KOL营销与内容种草领域的专业术语，主要在中国的内容电商平台（特别是小红书）上使用。指一种系统化、数据驱动的付费内容推广策略。

开白

开设白名单，一般在主体变更或开展新的项目、流程时会出现这个词。

口径

类比颗粒度。
• 在数据场景下：口径 = 计算逻辑 + 统计范围
• 在沟通场景下：口径 = 标准答案 + 统一说法

LTV

Life Time Value（或 Customer Lifetime Value），译为"用户生命周期价值"或"客户终身价值"。衡量一个用户在与品牌或产品建立关系的整个周期内，为企业带来的总收入或总利润。

NPL

NPL案例专栏，定位于内部案例的分享/交流。「N」代表笔记 「P」代表信任资产 「L」代表直播。一套基于过往运营经验总结得出的商家经营方法论框架。

年框

指广告公司与品牌方或平台之间签订的、期限为一年的战略性合作协议。"500w年框"是指在营销、广告、内容创作等领域内，年度合作总预算为500万元人民币的长期大单。

QIC

QIC质检服务是小红书平台为特定二手/全新奢侈商品提供的第三方质检服务（但其实源于抖音）。

透传

一般讲做"（不）给商家透传"，是指将上游（通常是平台、服务商或系统）的信息，原封不动、不加修改地直接传递给下游的商家（或不告知商家）。

一方回传

广告/数据领域最常用，指广告主/商家把自己第一方（自有）的用户行为、转化数据，主动回传给广告平台，用于优化广告模型、提升投放效果。

站外

这里的"站"通常指品牌完成最终转化和销售的核心阵地，最常见的是电商平台（如天猫、京东）或品牌的官方商城/APP；"站外"则指在上述销售场域之外的内容与社交平台。

战败人群

在企业的销售漏斗中，最终没有完成购买或转化的潜在客户。

七、Common Q&A

Q：商家申请"QIC加白"，要怎么操作？
A：不用操作，这个东西归【治理】操作，我们操作不了。
Q：佣金怎么收取，能否申请免佣？
A：梯度收取佣金，具体梯度见下表（来源于商家经营方法一本通），免佣计划参考小红书电商学习中心。注意百万免佣政策全平台性质的、是一次性的，一个主体只能享受一次，不是随随便便说申请就能申请的，在免佣计划结束之后就是阶梯佣金。
Q：为什么直播封面会自动更换？
A：设置直播计划的时候有系统自动生成的直播标题和封面，不改的话，直播的时候一关联直播计划就会自动变。
